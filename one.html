GitHub Copilot Features

Where can we run Copilot

  • Web (github.com)
  • CLI
  • IDE (VS Code, Intellij)
  • Chat (IDE, Web)

Editor based GitHub Copilot supporting

  • It explains the code in editor
  • It writes tests in editor
  • It fixes problems in editor
  • Generates Docs in editor
  • Majorly used for Generating repetitive code
  • Generating code from inline comments in natural language

Talk to your repository directly on the web

Chat participants:

  • @workspace
  • @github

Generate PR summary and commit message

Add file to chat

Questions can be on it:

  • What is the purpose of a particular method?
  • How to optimize certain code?

Guideline for Prompt engineering for GitHub Copilot

  • Avoid ambiguity
  • Indicate relevant code
  • Keep history relevant
  • Break complex tasks into simpler tasks
  • Organize your conversations with threads

Slash commands

Command Description Usage
/explain Get code explanations /explain what is the fetchPrediction method?
/fix Receive a proposed fix for the problems in the selected code /fix propose a fix for the problems in fetchAirports route
/tests Generate unit tests for selected code /tests
/help Get help on using Copilot Chat /help what can you do?
/clear Clear current conversation /clear
/doc Add a documentation comment /doc
/generate Generate code to answer your question /generate code that validates a phone number
/optimize Analyze and improve running time of the selected code /optimize fetchPrediction method
/new Scaffold code for a new workspace /new create a new django app
/simplify Simplify the selected code /simplify
/feedback Provide feedback to the team /feedback
